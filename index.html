ABOUT THE AZURE COSMOS DB BOOTSTRAP PROGRAM

Use this program to accelerate building or migrating your applications on Azure Cosmos DB.

Once you nominate your project and it gets approved, for 6 weeks, you will have access to the engineers in the Cosmos DB team, who would ensure that your application is built to achieve maximum performance at the lowest costs.

An Azure Cosmos DB engineer will be assigned to your project who would assist you in all the stages starting from project inception to production deployment.

At the end of this program, you will have a stellar application which follows the best-practices for accessing data in Azure Cosmos DB and is optimized to provide the best TCO and performance.
